Computer System

Computer consits of various hardware and software that provides the plateform for computer to function and perform different task is collectively known as computer system. (Hardware, Software, Data information, Procedure, User, Communication).

1. Hardware

Parts of computer that can be observed and touched. Physical device of the computer, equipment, input/output device, processing device and storage device.

2. Software

Use to describe the instruction that tells or instruct hardware for their corresponding perforemance.

3. Data/Information

1. Function of computer is to covert data into information.

2. Data consists of raw facts and figures that are processed to form information.

4. Procedure

It is a description of how thing or works are done, step for accomplishing a result. Reference manual is a guildlines, rules or instruction that provides the guidelines for processing the hardware of software.

5. Communication

Communication channel is required for sharing of the data or information between the computers. There are various manner on which various individual system are connected by wires, wireless cables, lines, etc.

Types of computer hardware.

Computer receive data, process it, produced output and stores it for further references such that computer should have four major components to perform these operation or tasks.

Input Device

It provides the platform for interaction with computer system or used enter data and instruction to the computer. These device convert input data and instruction to the suitable binary code like 0101 that can be accepted by the computer. Some major function of computer are:

Interaction with outside world and accept data & instruction.

Convert the received data & information into binary form and provide meaningful output.
