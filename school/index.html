Computer System

Computer consits of various hardware and software that provides the plateform for computer to function and perform different task is collectively known as computer system. (Hardware, Software, Data information, Procedure, User, Communication).

1. Hardware

Parts of computer that can be observed and touched. Physical device of the computer, equipment, input/output device, processing device and storage device.

2. Software

Use to describe the instruction that tells or instruct hardware for their corresponding perforemance.

3. Data/Information

1. Function of computer is to covert data into information.

2. Data consists of raw facts and figures that are processed to form information.

4. Procedure

It is a description of how thing or works are done, step for accomplishing a result. Reference manual is a guildlines, rules or instruction that provides the guidelines for processing the hardware of software.

5. Communication

Communication channel is required for sharing of the data or information between the computers. There are various manner on which various individual system are connected by wires, wireless cables, lines, etc.

Types of computer hardware.

Computer receive data, process it, produced output and stores it for further references such that computer should have four major components to perform these operation or tasks.

Input Device

It provides the platform for interaction with computer system or used enter data and instruction to the computer. These device convert input data and instruction to the suitable binary code like 0101 that can be accepted by the computer. Some major function of computer are:

1. Interaction with outside world and accept data & instruction.

2. Convert the received data & information into binary form and provide meaningful output.

3. Response for interpreting and executing most of the instruction and commands from the computer hardware and software and also control the operation of overall components of the computer system.

4. It receives the binary data from the input device and process according to the command and instruction and provides the results as output. It consists of the processing unit, memory unit and logical unit. Such that main function is to run the program by fetching instruction from the RAM, evaluating and executing them in sequence.

5. CPU reads instruction from memory, controlling the sequence of instruction and flow of data from one components to another.

6 .Communicating with various peripheral (input/output) using system bus.

Some major components of CPU are:

1. Control Unit

2. ALU

3. Register

Control Unit

1. Control unit provides the require timing and control signal for the operation of the computer. It controls the flow of data between the CPU, memory unit and input/output devices.

2. The instruction for the control unit is provided by the program stored in the main memory, interprets the instruction and issues the signals which cause the other unit of the system to execute them.

3. It also provides the status control, timing signal that is required for the operation of the various parts of the computer such that it is also know as central nervous system of the computer.

4. It stores program in the memory.

5. It fetches the instruction from the storage and decode each instruction and execute as required.

6. It gives commands to transfer data from the input device to memory to an ALU.

7. It transfers the result from ALU to memory and then to output devices.

ALU

Various computing functions are performed on data like addition, substraction, multiplication and devision and loglical operation such as comparison AND, OR and Excusive OR.

The result of the operation is stored in Accumulator or some register.

1. Accepts operands from register.

2. Perform arithmetic and logical operations.

3. Return a results to register or a memory.

4. Logical operation of ALU provides the computer to make decision ability.

Register

Register are high speed temporary memory storage locations in the CPU, made from the electronic devices like transistors, flip flop, etc.
